we are trying to make internboat happen.

why did dis? (purpose)

internships take forever to apply to when you have to access each company website and fill out all their forms, over and over and over. we wanted to make this easy by localizing all the applications in one place.

how do dat? (structural process)

  • html | for front-end development
  • css | for making it not ugly
  • javascript | for writing the program back-end
  • firebase auth | for storing user data
  • firebase database | for storing application data
  • web scraping | to get additional application requirements
